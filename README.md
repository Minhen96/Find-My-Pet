# 🐾 Find-My-Pet

A community-driven lost and found pet platform designed for efficiency, speed, and reliability.

## 🏗️ Project Structure

- **[backend](./backend)**: NestJS API with PostgreSQL/PostGIS.
- **[mobile](./mobile)**: Flutter application for iOS and Android.

## 🚀 Phase 1 Progress: Foundations
- [x] Backend Global Standards (Exceptions, Middleware, Interceptors)
- [x] Secure JWT Authentication with Refresh Tokens
- [x] Brute-Force Protection (Rate Limiting)
- [x] Geolocation-ready Database (PostGIS)
- [x] Flutter App Setup & Navigation

## 🛠️ Quick Start

1. Start the database: `docker-compose up -d`
2. Run the backend: `cd backend && npm run start:dev`
3. Run the mobile app: `cd mobile && flutter run`
